Tentang Software

Software (perangkat lunak) adalah sekumpulan program atau perintah di komputer dan HP yang berfungsi untuk menjalankan tugas tertentu.

Jika hardware adalah benda fisik seperti keyboard, layar, dan mouse, maka software adalah bagian yang membuat perangkat bisa digunakan.

Fungsi Software

  • Mengoperasikan komputer atau HP
  • Membantu pekerjaan seperti mengetik, edit foto, dan desain
  • Menjalankan game atau aplikasi
  • Mengatur sistem agar perangkat berjalan normal

Jenis-Jenis Software

1. Software Sistem

Software sistem adalah perangkat lunak yang mengatur jalannya perangkat.

  • Windows
  • Android
  • iOS
  • Linux

2. Software Aplikasi

Software aplikasi adalah perangkat lunak yang digunakan pengguna untuk melakukan aktivitas tertentu.

  • WhatsApp
  • Instagram
  • Google Chrome
  • Microsoft Word
  • CapCut

3. Software Utility

Software utility adalah perangkat lunak yang membantu perawatan sistem.

  • Antivirus
  • Pembersih file sampah
  • Backup data
